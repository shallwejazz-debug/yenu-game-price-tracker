## 보류 항목 (기록만 해두고 대기 중)

### 1. 검색창 위치 이동
- 우선순위: 🟡 중
- 예상 작업: 🟢 소 (10분)
- 내용: 페이지 최상단(제목 줄) → 우측 "특가 순위" 박스 **바로 위**로 이동. 페이지 상단이 아니라 사이드바 칼럼 안 상단.
- 상태: ⬜ 대기

### 2. 썸네일 전문 규격 교체
- 우선순위: 🔴 상 (사이트 신뢰도 직결)
- 예상 작업: 🟡 중 (Steam/IGDB API 연동 필요 — 5번과 통합)
- 내용: 네이버 쇼핑 이미지(패키지 박스/합쳐진 이미지/흰 배경) → Steam 공식 header / IGDB 커버로 통일. 호라이즌·디아블로가 특히 지저분함.
- 상태: ⬜ 대기 (5번과 함께 처리 권장)

### 3. 모바일 사이드바 처리
- 우선순위: 🔴 상 (사용자 70%가 모바일)
- 예상 작업: 🟡 중 (CSS 미디어쿼리 + 레이아웃 재구성)
- 내용: 모바일에서 우측 특가 순위 사이드바가 좁아 카드가 답답함 → **상단 가로 스크롤 띠** 또는 **하단 별도 섹션**으로 이동.
- 상태: ✅ 1차 처리됨 (이번 세션: 모바일에서 특가순위를 본문 아래 별도 섹션으로 분리)

### 4. 특가 순위 정가 호버 툴팁
- 우선순위: 🟢 하
- 예상 작업: 🟢 소 (10분)
- 내용: -70%인데 정가가 안 보임 → 호버 시 "정가 ₩66,000 → ₩19,800" 툴팁 표시.
- 상태: ✅ 처리됨 (이번 세션)

### 5. 디지털 가격 / 이미지 / 메타데이터 자동화 (가장 큰 효과)
- 우선순위: 🔴 상 (데이터 완성도 + 신뢰도 최대 효과)
- 예상 작업: 🔴 대 (3~5일, 단계 분할 진행)
- 다단계 폴백 흐름:
  ```
  게임명 입력
   → 1차: Steam API (게임명 → appid → 공식 이미지 + 정가 + 한글 제목 + 디지털 가격)
   → 2차: RAWG API (멀티플랫폼 검색 → 공식 이미지)
   → 3차: IGDB API (백업)
   → 폴백: 관리자 수동 이미지 URL / appid 직접 입력
  ```
  → PC 게임 95% + 콘솔 게임 90% 자동, 나머지 5~10%만 수동.
- 구현 팁:
  - **이미지 캐싱 필수**: Steam CDN 직링 금지 → Cloudflare R2(무료 10GB)에 `{game_id}_header.jpg` 저장 (CDN 정책 변경 대비).
  - **비율 통일**: 카드용 460×215(header), 상세용 600×900(library_600x900).
  - **한글 제목**: Steam API `l=korean` 파라미터 → 한글 매칭, 실패 시 영어 폴백.
  - **Battle.net 예외**: 디아블로3 = Steam 없음, 디아블로4 = 2344520(Battle.net 전용). RAWG/IGDB 폴백 or 수동.
  - **관리자 콘솔 보강**: 미리보기에서 이미지 후보 3~5장 선택 / URL 수동입력 / appid 직접입력 옵션.
- appid 시드값 (자주 쓸 게임):
  ```
  엘든 링            1245620
  발더스 게이트 3      1086940
  사이버펑크 2077      1091500
  스타듀 밸리          413150
  호라이즌 포비든 웨스트 2420110
  디아블로 2 레저렉션   1928670
  디아블로 4           2344520  (Battle.net 전용)
  디아블로 3           Steam 없음 (Battle.net 전용)
  ```
- 세부 우선순위: ① Steam API 연동(1~2일, 효과 최대) → ② R2 이미지 캐싱(반나절) → ③ RAWG 연동(1일, 콘솔 보강) → ④ 콘솔 이미지 미리보기/수동입력 폴백(반나절)
- 상태: 🔄 1단계 진행 중 (이번 세션: Steam API 디지털 가격 자동 수집 시작)

### 6. 메인 페이지(/) 랜딩 분리
- 우선순위: 🟡 중 (첫 방문자 전환율 영향)
- 예상 작업: 🟡 중 (반나절)
- 내용: 현재 `/`와 `/games`가 동일 페이지 → 메인을 **랜딩 페이지**로 분리. "이번 주 핫딜 Top 5 + 알림 가입 CTA + 사이트 소개".
- 상태: ⬜ 대기
